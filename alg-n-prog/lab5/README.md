# Структуры данных: деревья (Практическая работа №5)

Практическая работа №5 по дисциплине "Основы алгоритмизации и программирования".

---

## 📌 Описание

В данной практической работе изучаются основные алгоритмы работы с деревьями. Рассматриваются понятия бинарного дерева, дерева поиска, способы обхода деревьев (прямой, обратный, концевой, в ширину), а также основные операции: поиск, включение, удаление элементов, сравнение и копирование деревьев.

---

## ⚙️ Функциональность

- Построение бинарного дерева из списочной записи
- Подсчет листьев при различных обходах
- Поиск элемента с выводом пройденных узлов
- Определение длины пути до заданного элемента
- Подсчет вершин на заданном уровне
- Поиск одинаковых элементов
- Печать дерева по уровням (обход в ширину)
- Работа с деревом-формулой (вычисление, проверка, преобразования)
- Построение и преобразование деревьев поиска

### Задание 1. Подсчет листьев в бинарном дереве

**Постановка задачи:**
В заданном бинарном дереве подсчитать число его листьев и напечатать их значения при различных способах обхода.

**Решение:**
Программа выполняет обход дерева четырьмя способами:
- **а) Прямой обход (итеративно):** используется стек, узлы посещаются в порядке корень-лево-право
- **б) Обратный обход (итеративно):** используется стек, узлы посещаются в порядке лево-корень-право
- **в) Концевой обход (итеративно):** используется два стека, узлы посещаются в порядке лево-право-корень
- **г) Рекурсивный обход:** функция вызывает себя для левого и правого поддеревьев

### Задание 7. Дерево-формула

**Постановка задачи:**
Формулу вида `<формула> ::= <терминал> | (<формула> <знак> <формула>)` можно представить в виде двоичного дерева.

**Решение:**
- **а) Построение и вычисление:** по заданной формуле строится дерево, выполяются три вида обхода и вычисляется значение
- **б) Проверка на формулу:** проверяется, является ли заданное дерево корректной формулой (листья — цифры, внутренние узлы — операторы)
- **в) Дистрибутивность:** преобразование `(a±b)*c` → `(a*c)±(b*c)`
- **г) Обратное преобразование:** преобразование `(a*c)±(b*c)` → `(a±b)*c`

### Задание 8. Дерево поиска из букв

**Постановка задачи:**
