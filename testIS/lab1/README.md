# Калькулятор квадратных уравнений

Лабораторная работа №1 по дисциплине "Тестирование Информационных Систем".

---

## 📌 Описание

Простое веб-приложение для решения квадратных уравнений вида **ax² + bx + c = 0** с набором тестовых случаев.

---

## ⚙️ Функциональность

- Решение квадратных уравнений
- Обработка особых случаев (линейные уравнения, a=0 и т.д.)
- 7 тестовых случаев из лабораторной работы
- Простой и понятный интерфейс

## 🧪 Тестовые случаи

| № | Проверяемый случай | a | b | c  | Ожидаемый результат       |
|---|--------------------|---|---|----|---------------------------|
| 1 | D > 0              | 1 | 1 | -2 | x₁ = 1, x₂ = -2           |
| 2 | D = 0              | 1 | 2 | 1  | x₁ = x₂ = -1              |
| 3 | D < 0              | 2 | 1 | 2  | Нет действительных корней |
| 4 | a=0, b=0, c=0      | 0 | 0 | 0  | x - любое число           |
| 5 | a=0, b=0, c≠0      | 0 | 0 | 2  | Нет решения               |
| 6 | a=0, b≠0           | 0 | 2 | 1  | x = -0.5                  |
| 7 | a≠0, b≠0, c=0      | 2 | 1 | 0  | x₁ = 0, x₂ = -0.5         |

---

## 🚀 Как запустить

1. Скачайте все файлы
2. Откройте `index.html` в браузере
3. Введите коэффициенты и нажмите "Решить уравнение"
4. Нажмите "Запустить тесты" для проверки всех случаев
