# Лабораторная работа №2: Тестирование «белым ящиком»

Лабораторная работа №2 по дисциплине "Тестирование Информационных Систем".

---

## 📌 Описание проекта
Данный проект представляет собой решение лабораторной работы по теме "Тестирование «белым ящиком»". 
В рамках работы разработаны программы для определения вида треугольника по сторонам и углам, 
а также проведено их тестирование с использованием методологии «белого ящика».

## 🎯 Цель работы
Изучить метод тестирования «белым ящиком», научиться разрабатывать тестовые сценарии на основе 
анализа внутренней структуры программы, выявлять дефекты и формулировать рекомендации по их исправлению.
