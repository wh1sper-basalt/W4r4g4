# Репозиторий учебных проектов W4r4g4

Welcome в мой репозиторий с лабораторными и практическими работами. Здесь хранятся проекты, выполненные в ходе обучения по трем дисциплинам, связанным с разработкой и тестированием программного обеспечения.

---

## 📁 Структура репозитория

Репозиторий разделен на три основные папки, соответствующие предметам:

### 1. Разработка кода информационных систем (`/RKIS`)

В данной папке представлены практические работы по созданию компонентов информационных систем. Здесь больше внимания уделяется фронтенд- и бэкенд-составляющим приложений.

* **Используемые технологии:** `HTML`, `CSS`, `JavaScript`.

* **Содержание:** Верстка пользовательских интерфейсов, написание клиентской логики, создание интерактивных веб-страниц.

### 2. Основы алгоритмизации и программирования (`/alg-n-prog`)

В этой папке находятся работы, направленные на изучение базовых алгоритмов, структур данных и основ императивного программирования.

* **Используемые технологии:** Основной упор делается на `C++` и `Python`.

* **Содержание:** Реализация классических алгоритмов (сортировки, поиск, работа с массивами и строками), решение задач на алгоритмическое мышление.

### 3. Тестирование информационных систем (`/testIS`)

Этот раздел посвящен изучению методологий и инструментов тестирования. Здесь собраны проекты, демонстрирующие процесс проверки работоспособности и качества кода.

* **Используемые технологии:** `Python`, `JavaScript`.

* **Содержание:** Модульные тесты, интеграционные тесты, написание тест-кейсов, тестирование веб-интерфейсов (в связке с HTML/CSS/JS).

---

## 🛠️ Стек технологий

Весь код в репозитории написан с использованием следующих языков и инструментов:

* **Frontend:** HTML5, CSS3, JavaScript

* **Backend/Algorithms:** Python, C++

---

## 🎯 Цель репозитория

Репозиторий создан для систематизации выполненных работ, отслеживания прогресса в учебе, а также для демонстрации навыков программирования и тестирования. Код может быть полезен как шпаргалка или пример выполнения типовых заданий.

---

## 🚀 Навигация

Для просмотра конкретной работы перейдите в соответствующую папку:

*   `alg-n-prog/` — Основы алгоритмизации и программирования.

*   `testOS/` — Тестирование информационных систем.

*   `RKIS/` — Разработка кода информационных систем.
