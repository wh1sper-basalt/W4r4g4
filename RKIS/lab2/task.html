Практическая работа №2

Bootstrap: расположение блоков

Задание 1. Три в ряд

Условие:

Создайте три блока и расположите их в ряд.

Emmet сокращение:

<!-- Три блока в ряд с помощью Bootstrap -->
.container>.row>.col*3>.demo-block{Блок $}

Решение:

<div class="container">
  <div class="row">
    <div class="col"><div class="demo-block">Блок 1</div></div>
    <div class="col"><div class="demo-block">Блок 2</div></div>
    <div class="col"><div class="demo-block">Блок 3</div></div>
  </div>
</div>
Блок 1
Блок 2
Блок 3
Пояснение: Использована сетка Bootstrap с классом .row и тремя колонками .col-md-4, которые занимают равную ширину на средних устройствах и встают в столбик на мобильных.

Задание 2. Большой контейнер

Условие:

Создайте контейнер, который будет растянут на всю ширину экрана. Выведите в этом контейнере заголовок первого уровня: "Привет мир".

Emmet сокращение:

<!-- Контейнер на всю ширину -->
.container-fluid>h1{Привет мир}

Решение:

<!-- Контейнер на всю ширину экрана -->
<div class="container-fluid bg-light p-4 border rounded">
  <h1 class="text-center display-4">Привет мир</h1>
</div>

Привет мир

Пояснение: Использован класс .container-fluid, который занимает 100% ширины экрана, в отличие от обычного .container с фиксированной шириной.

Задание 3. Расположение блоков

Условие:

Создайте несколько блоков на сайте и расположите их так, как показано на картинке.

Emmet сокращение:

<!-- Сложное расположение блоков -->
.row>.col-6*2^.row>.col-8+.col-4

Решение:

<!-- Первый ряд: два блока по 50% -->
<div class="row mb-3">
  <div class="col-6"><div class="demo-block">Блок 1 (50%)</div></div>
  <div class="col-6"><div class="demo-block">Блок 2 (50%)</div></div>
</div>

<!-- Второй ряд: 66.6% и 33.3% -->
<div class="row">
  <div class="col-8"><div class="demo-block">Блок 3 (66.6%)</div></div>
  <div class="col-4"><div class="demo-block">Блок 4 (33.3%)</div></div>
</div>
Блок 1 (50%)
Блок 2 (50%)
Блок 3 (66.6%)
Блок 4 (33.3%)
Пояснение: Использованы разные комбинации колонок Bootstrap: .col-6 (50%), .col-8 (66.6%), .col-4 (33.3%) для создания сложной структуры.

Задание 4. Вертикальное расположение блоков

Условие:

При помощи встроенных классов Bootstrap расположите блоки таким образом, чтобы они находились вверху родительского блока, по центру, а также внизу.

Emmet сокращение:

<!-- Вертикальное выравнивание -->
.row.align-items-start>.col*3
.row.align-items-center>.col*3
.row.align-items-end>.col*3

Решение:

<!-- Верхнее выравнивание -->
<div class="row align-items-start" style="min-height: 200px;">
  <div class="col"><div class="demo-block">Верх</div></div>
  <div class="col"><div class="demo-block">Верх</div></div>
  <div class="col"><div class="demo-block">Верх</div></div>
</div>
Выравнивание по верхнему краю (align-items-start)
Верх
Верх
Верх
Выравнивание по центру (align-items-center)
Центр
Центр
Центр
Выравнивание по нижнему краю (align-items-end)
Низ
Низ
Низ
Пояснение: Использованы классы Bootstrap для вертикального выравнивания: .align-items-start, .align-items-center, .align-items-end.

Задание 5. Отступы между блоками

Условие:

Создайте два блока и добавьте между ними отступ. Общая ширина одного этого ряда должна быть 100% ширины экрана.

Emmet сокращение:

<!-- Блоки с отступами -->
.container-fluid>.row.g-4>.col*2>.demo-block

Решение:

<!-- Два блока с отступом на всю ширину -->
<div class="container-fluid px-0">
  <div class="row g-4 mx-0">
    <div class="col-md-6"><div class="demo-block">Блок 1</div></div>
    <div class="col-md-6"><div class="demo-block">Блок 2</div></div>
  </div>
</div>

Блок 1

Этот блок занимает половину ширины экрана

Блок 2

Этот блок занимает половину ширины экрана

Демонстрация разных отступов (gutter):

Маленький отступ (g-2):

Блок с отступом 0.5rem
Блок с отступом 0.5rem

Средний отступ (g-4):

Блок с отступом 1.5rem
Блок с отступом 1.5rem

Большой отступ (g-5):

Блок с отступом 3rem
Блок с отступом 3rem
Пояснение: Для отступов между колонками используются классы g-* (gutter). Общая ширина ряда всегда 100% благодаря .row и .col-*.
